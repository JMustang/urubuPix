# Commands

- Comando para iniciar o Gemfile.

```bash
bundle init
```

- Gems usadas nessa api.

```Gemfile
gem 'dotenv'
gem 'puma'
gem 'rake'

# Framework, equivalente ao express.js do node, usado no projeto
gem 'sinatra'
gem 'sinatra-activerecord'

# Banco de dados usado no projeto
gem 'sqlite3'
```
